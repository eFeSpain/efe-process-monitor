# Capturas para el README

Las capturas no se pudieron generar automáticamente, así que añádelas tú (30 segundos). Con la
app abierta, usa **Win + Mayús + S** (Windows) para recortar y guarda cada PNG en esta carpeta
`docs/` con **el nombre exacto** de la tabla. El README ya las referencia.

| Archivo | Qué capturar |
|---------|--------------|
| `screenshot-main.png` | El panel principal: cabecera + feed de actividad en vivo + la tabla de conexiones ordenada por riesgo. **Es la imagen estrella** — hazla ancha y con datos. |
| `screenshot-details.png` *(opcional)* | Una fila desplegada, mostrando el bloque de Proceso, la inteligencia de la IP remota (con el chip de proveedor / desglose del riesgo) y los botones de acción. |
| `screenshot-capture.png` *(opcional)* | Un panel de captura `tshark` en vivo con paquetes (líneas TLS SNI / HTTP / DNS). |
| `screenshot-history.png` *(opcional)* | El modal de Histórico o el panel de IPs bloqueadas. |

Consejos para una buena imagen principal:
- **Maximiza** la ventana del navegador antes de recortar (cuanto más ancha, más profesional).
- Ten alguna conexión con riesgo distinto de cero para que se vean las insignias de colores.
- Pon el idioma según tu público (inglés para el README global; ES/EN con el botón de arriba).
- Si recortas el navegador entero, oculta antes pestañas/barras personales para que quede limpio.
